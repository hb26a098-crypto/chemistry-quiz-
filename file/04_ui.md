# 4. 画面遷移とUI設計

## 4.1. 画面構成一覧
本ゲームは以下の3つの主要画面で構成され、シンプルな画面遷移（シングルページアプリケーション形式）で設計します。

1. **タイトル画面 (Title Screen)**
2. **クイズ画面 (Quiz Screen)**
3. **結果画面 (Results Screen)**

---

## 4.2. 各画面のUI構成とレイアウト

### 4.2.1. タイトル画面
* **レイアウト**:
  * 画面中央にゲームタイトル「化学クイズゲーム」を大きく配置。
  * その下に、プレイモードを選択する2つのボタンを縦または横に並べて配置する。
* **ボタン要素**:
  * **「通常モードで開始」ボタン**: 活性状態。クリックすると通常クイズが開始される（無機4問、有機4問の計8問を完全シャッフルして出題）。
  * **「苦手問題モードで開始」ボタン**:
    * 苦手問題リストに1件以上の登録がある場合: 活性状態。クリックすると苦手問題の連続出題が開始される。
    * 苦手問題リストが空（0件）の場合: **非活性（グレーアウト）**となり、クリックできない。ボタン付近に「現在、苦手問題はありません」等の補助テキストを表示する。

### 4.2.2. クイズ画面
* **レイアウト (上から順に配置)**:
  1. **上部: 進捗インジケーター**: 
     * 現在の進捗状況（例: 「3 / 8 問目」または苦手モード時は「2 / 5 問目」など）を分かりやすく表示。
  2. **問題表示エリア**:
     * 出題文（例: 「二酸化硫黄の化学式を答えよ」など）を大きく見やすいフォントで表示。
  3. **中央: 解答エリア**:
     * **4択問題の場合**: 縦並びの4つの選択肢ボタン。
     * **○×問題の場合**: 「○（正しい）」と「×（誤り）」の2大ボタン。
     * **記述式問題の場合**: テキスト入力欄（プレースホルダー「物質名または化学式を入力」）と、「解答する」ボタン。
  4. **回答後フィードバック・解説エリア (解答確定後に出現)**:
     * 解答ボタン選択または入力後に、その場で即座に表示される。
     * 「正解！」（緑系）または「不正解」（赤系）のわかりやすいフィードバックメッセージ。
     * 正解値の表示と、問題ごとの「解説文」。
     * 最下部に **[次へ] ボタン**（次の問題、または全問終了時は結果画面へ遷移するためのボタン）。

### 4.2.3. 結果画面
* **レイアウト**:
  * 画面中央に「結果発表」と最終成績をダイナミックに表示。
  * **最終スコア表示**: 「○ 問正解 / 8 問中」（苦手問題モードの場合は「○ 問正解 / ○ 問中」）と表示。
  * **ボタン要素**:
    * **「タイトルに戻る」ボタン**: クリックすると、すべてのクイズ状態をクリアして「タイトル画面」へ戻る。
