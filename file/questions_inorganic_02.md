# 無機化学 問題リスト02 (第51問〜第100問)

本ファイルは、無機化学のうち**「金属元素とその化合物」**（アルカリ金属、アルカリ土類金属、両性金属、遷移金属、および金属イオンの系統分離・呈色反応など）に関する50問を収録した問題マスターデータです。

## 1. 4択問題 (25問)

### 【問51】
* **ID**: `INORG_02_051`
* **問題タイプ**: 4択 (FourChoice)
* **問題文**: 炎色反応において、「赤紫色」を示す元素はどれか？
* **選択肢**:
  * [0] リチウム
  * [1] カリウム
  * [2] ナトリウム
  * [3] カルシウム
* **正解**: `1` (カリウム)
* **解説**: 各元素の炎色反応は、リチウムが赤、ナトリウムが黄、カリウムが赤紫、カルシウムが橙赤です。

### 【問52】
* **ID**: `INORG_02_052`
* **問題タイプ**: 4択 (FourChoice)
* **問題文**: 炭酸ナトリウム $Na_2CO_3$ を工業的に製造するソルベー法（アンモニアソーダ法）の出発原料（主原料）の組み合わせとして正しいものはどれか？
* **選択肢**:
  * [0] 塩化ナトリウム と 石灰石
  * [1] 水酸化ナトリウム と 二酸化炭素
  * [2] 硫酸ナトリウム と 炭酸アンモニウム
  * [3] 塩化カルシウム と アンモニア
* **正解**: `0` (塩化ナトリウム と 石灰石)
* **解説**: ソルベー法の主原料は、塩化ナトリウム $NaCl$ （飽和食塩水）と、二酸化炭素および酸化カルシウムの原料となる石灰石 $CaCO_3$ です。

### 【問53】
* **ID**: `INORG_02_053`
* **問題タイプ**: 4択 (FourChoice)
* **問題文**: 2族元素（アルカリ土類金属およびその同族）のうち、単体が「常温の水」と反応しない金属はどれか？
* **選択肢**:
  * [0] マグネシウム
  * [1] カルシウム
  * [2] ストロンチウム
  * [3] バリウム
* **正解**: `0` (マグネシウム)
* **解説**: カルシウム、ストロンチウム、バリウムは常温の水と激しく反応して水素を発生しますが、マグネシウム $Mg$ は常温の水とはほとんど反応せず、熱水と反応して水素を発生します（ベリリウムは熱水とも反応しません）。

### 【問56】

* **問題タイプ**: 4択 (FourChoice)
* **問題文**: 水酸化アルミニウム $Al(OH)_3$ の沈殿に過剰の「水酸化ナトリウム水溶液」を加えたときに起こる変化として正しいものはどれか？

### 【問58】
* **ID**: `INORG_02_058`
* **問題タイプ**: 4択 (FourChoice)
* **問題文**: 亜鉛イオン $Zn^{2+}$ を含む酸性水溶液に硫化水素 $H_2S$ を通した際の変化として正しいものはどれか？
* **選択肢**:
  * [0] 白色の硫化亜鉛 $ZnS$ が沈殿する
