# 1. ゲーム概要と動作環境

## 1.1. ゲーム概要
* **ジャンル**: 化学クイズゲーム
* **対象分野**: 「無機化学」および「有機化学」
* **開発コンセプト**:
  * じっくり学習できる制限時間・ヒントなしのクイズシステム。
  * 繰り返し間違えた問題を効率的に復習できる「苦手問題モード」を搭載。

## 1.2. 動作環境・対応デバイス
* **対象デバイス**: スマートフォンおよびPCブラウザ
* **対応要件**:
  * レスポンシブ対応（画面サイズに応じてレイアウトが最適化されること）
  * タッチ操作（スマートフォン）およびクリック操作（PC）の両方で快適にプレイ可能であること
