# 2. ゲームシステムとメカニクス

## 2.1. 基本プレイフロー
1. **分野選択**: プレイヤーは「通常プレイ」または「苦手問題モード」を選択する。
2. **通常プレイ**:
   * 無機化学から4問、有機化学から4問の計8問が選ばれる。
   * **出題順序**: 無機・有機の分野が完全に混ざり合った状態（完全なランダムシャッフル）で、合計8問が順に出題される。
3. **結果発表**: 8問すべての回答終了後、正解数や結果を表示する。

## 2.2. 回答形式
1分野あたり4問の出題内訳は以下の通りとする。

* **4択問題 (2問)**: 4つの選択肢から1つの正解を選択する。
* **○×問題 (1問)**: 提示された説明文が正しい（○）か誤り（×）かを選択する。
* **記述式問題 (1問)**: キーボードで物質名（例: 「二酸化炭素」「エタノール」など）や化学式（例: 「CO2」「C2H5OH」など）を直接テキスト入力して解答する形式。
  * **入力の自動補正ルール (判定時の正規化)**: 正誤判定の際、システムは以下の自動補正を行ってから判定する。
    1. 全角文字から半角文字への変換（アルファベット、数字など）
    2. アルファベットの「大文字・小文字」の区別をなくす（すべて小文字に統一して比較）
    3. 入力されたテキストから「不要なスペース（前後および文字間の空白）」を完全に除去

## 2.3. 苦手問題モード
間違えた問題を自動的に記録し、集中的に再挑戦できるモード。

* **自動登録条件**: 
  * 過去のすべてのプレイを通じて、同じ問題を通算（累計）で5回間違えた時点で、その場で苦手問題リストへ自動登録される。
  * 誤答回数はセーブデータとして永続的にカウントが保持される。
* **苦手問題モードでの出題**:
  * 苦手問題リストに登録されている全問題が出題される（出題数の制限なし）。
* **リストからの解除条件**:
  * 苦手問題モードでプレイし、該当する苦手問題に「正解」した時点で、自動的に苦手リストから解除（削除）される。
  * 解除された際、該当問題の通算誤答回数カウントはリセット（0回にクリア）される。
* **苦手問題がゼロのときの挙動**:
  * 苦手問題リストに登録されている問題が1件もない場合、タイトル画面の「苦手問題モードで開始」ボタンは**非活性（グレーアウト）**となり、クリックできない状態にする。

## 2.4. 解答時のフィードバックと進行フロー
* 1問解答するごとに、その場（回答直後）で「正解」または「不正解」のフィードバックメッセージを画面に表示する。
* 同時に、問題に設定されている「解説文」を画面に表示し、プレイヤーが学習できるようにする。
* プレイヤーは表示された解説を確認した後、画面の **[次へ] ボタン** を押すことで、次の問題へ進む。
